# Product Hunt Launch

## Tagline
Queue X posts in bulk — close the tab, we publish on time.

## Description
XQueue lets you bulk-paste X posts with scheduled times. A cloud worker publishes via the official X API even when your browser is closed. Free tier: 10 text posts/month. Pro: 150 posts + images for $15/mo.

## First comment (maker)
Built this because I was tired of leaving browser tabs open just to hit "post" at the right time. No sketchy Chrome extensions — just OAuth + official API. Would love feedback on bulk paste UX!
